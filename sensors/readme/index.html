README

/temperature

"units="
"C" (default)
"F"
"K"
"duration= [seconds]"
"interval= [seconds]"

Example: /sensors/temperature/index.php?units=C&duration=60&interval=5

(will return a temperature reading from the DS18b20 module every 5 seconds for 60 seconds in degrees Celcius)

/barometer

"tunits="
"C" (default)
"F"
"K"
"punits="
"Pa" (default)
"atm"
"PSI"
"aunits="
"m" (default)
"ft"
"duration= [seconds]"
"interval= [seconds]"

Example: /sensors/barometer/index.php?tunits=f&punits=psi&aunits=ft&duration=60&interval=5

(will return temperature, pressure, and altitude readings from the BMP180 module every 5 seconds for 60 seconds in degrees Fahrenheit, pounds per square inch, and feet respectively)

/ultrasonic

"units="
"cm" (default)
"in"
"duration= [seconds]"
"interval= [seconds]"

Example: /sensors/ultrasonic/index.php?units=in&duration=60&interval=5

(will return a distance reading from the ultrasonic ranging module every 5 seconds for 60 seconds in degrees Celcius)
